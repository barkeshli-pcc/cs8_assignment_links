Binary Search Tree (BST)

implement the following tree_node structure:

template <class T>
struct node{
    node(T n);
    T item;
    node<T>* left;
    node<T>* right;
};

this is a node that has a left and right pointers.

Now, based on the rules of the binary search tree, design a print,  an insert and a boolean search function.

The print function prints the right  subtree first, then the root, then the left subtree:

        [76]
    [71]
        [51]
[27]
        [19]
    [9]

 

Write an interactive function that will allow the user to choose to insert a number, insert a random number, or search for a number to see if the tree contains that number. Print the tree after each exchange.
