The Lab and Plane problem from CS3.

In three levels. Application, Lab and Plane Functions, Basic 2D array Functions:

 

We will write a set of functions to make use of two dimensional dynamic arrays simpler:

int** _allocate(int sizes[]);

bool _index_is_valid(sizes[], int row, int column)

functions for reading and writing into a two dimensional dynamic array (this could be done by simply addressing the cells with the bracket operator. But we are not using that simple approach because we need to be very comfortable using pointers.)

You may want to write a function that returns a pointer to a given cell in the 2D array: 

int** _get(int** two_d, int *sizes, int row, int column)
