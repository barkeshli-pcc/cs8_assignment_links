Don't forget the proper use of function objects in these projects:

 

1. the Map class: A unique sorted list of Pair objects:

Insert function: If duplicate entries are inserted, the function will ignore the inserted item.

[ ]: The bracket operator: When duplicated entries are inserted, the function will replace the existing item with the new.

 

2. Frequency: A unique sorted list of Pairs of templated items and ints:

Insert Function: When duplicated entries are inserted, the function will increment the int value of the pair.

[ ]: The bracket operator returns the value of the key: cout<<F["Complicated"]; //return the frequency of the key "Complicated"

 

Don't forget the usual functions for printing, deleting, Merging, etc. See the STL map functions for ideas. You have already written all the necessary functions for these.
